# Entity Relationship Diagram

### Keys

Like data base system that must have a primary key

- `PK` is Primary Key
- `FK` is Foreign Key

### Cardinality

|Cardinality|Meaning|
|--|--|
|`\|\|`|Exactly one|
|`\|o`|Zero or one|
|`}\|`|One or more|
|`}o`|Zero or more|

### Line

|Line|Meaning|
|--|--|
|`--`|Identifying Relationship|
|`..`|Non-identifying Relationship|
